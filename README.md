# mini-E-Commerce - ECommerce System


## Features

### Public Features
- Browse all available products
- Filter products 

### Customer Features
- Register and login as customer
- Add meals to cart
- Place orders (Cash on Delivery)
- Track order status
- Manage profile


### Admin Features
- View all users (customers and providers)
- Manage user status (suspend/activate)
- View all orders

## Technology Stack

- Node.js + TypeScript
- Express.js (web framework)
- PostgreSQL (database)
- bcrypt (password hashing)
- jsonwebtoken (JWT authentication)
- betterAuth

## Setup & Usage Instructions

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/rajon38/mini_E-Commerce.git
cd mini_E-Commerce
npm install
```

### Usage

```bash
npm run dev
```

Visit `http://localhost:8020` in your browser for local server.
